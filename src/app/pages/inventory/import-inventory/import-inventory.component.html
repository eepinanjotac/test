Instrucciones

Siga las instrucciones cuidadosamente antes de importar el archivo.
Las columnas del archivo CSV deben estar en el siguiente orden.
Número de Columna Nombre de Columna Instrucciones
1 SKU Código de producto
2 NAME Nombre del producto
3 DESCRIPTION Descripción del Producto
4 SPECIFICATION Especificaciones Técnicas
5 PUR_PRICE Precio compra
6 MIN_STOCK Alerta de stock
7 STOCK Stock actual
